ZM Connector simplifies the process of creating attractive and effective landing pages for Amazon affiliates.

== Description ==

ZM Connector is a WordPress plugin designed to help Amazon affiliates create engaging landing pages with ease. By connecting directly to a backend service, the plugin allows users to generate content dynamically, incorporating product details, images, and affiliate links without manual input. This tool is perfect for affiliates looking to increase their conversion rates and streamline their content creation process.

Key Features:
- **Automated Landing Page Creation**: Generate landing pages for Amazon products with just a few clicks.

- **SEO Friendly**: Optimized for search engines to help you rank better and attract more traffic.
- **Responsive Design**: Ensures your landing pages look great on any device.

== Installation ==

1. Upload the `zm-connector` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Go to the ZM Connector menu item in your admin sidebar to configure the plugin.

== Frequently Asked Questions ==

= Do I need an Amazon affiliate account to use ZM Connector? =

Yes, you need to have an Amazon affiliate account since the plugin is designed to create landing pages with your affiliate links embedded.

= Can I customize the layout of the landing pages? =

While the initial version offers limited customization from the WordPress admin, future versions are planned to include more extensive layout and styling options.

= Is ZM Connector free to use? =

Yes, the plugin is free to use, but it requires an active subscription to the backend service for landing page generation.
